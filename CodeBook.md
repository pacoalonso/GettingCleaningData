Code Book of the _tidyData.txt_ file
------------------------------------


* __subject__: Each one of the 30 subjects who participated in the project coded by numbers

* __activity__:   Each one of the activities analyzed
    1. WALKING
    2. WALKING_UPSTAIRS
    3. WALKING_DOWNSTAIRS
    4. SITTING
    5. STANDING
    6. LAYING
